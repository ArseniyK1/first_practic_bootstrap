теги html
html - корневой тег, в котором находятся остальные теги
head - используется, чтобы добавлять мету - информацию, которая описывает html страницу (ссылки на таблицы стилей и тп)
body - в него добавляется остальные теги a, p, div и тд.
a - в тег а можно добавлять ссылки(внутренние и внешние)
h1 - заголовок(должен быть один на 1 хтмл странице)
p - в тег р можно добавлять большие блоки текста , которые состоят из нескольких предложений
meta (внутри тега head) - мета инф. которая описывает страницу
link - таблица стилей
style - для подключения стилей css
script - для подключения js кода
